# Mobile Posting Guide

## Quick Start - Post from Any Device

You already have a complete posting dashboard set up! Here's how to use it:

### 1. Access Your Admin Panel

**URL:** https://enrichmetwo.win/admin/

- Works on any device (phone, tablet, computer)
- Bookmark this URL on your phone for quick access
- Add to home screen on mobile for app-like experience

### 2. First Time Login

1. Go to https://enrichmetwo.win/admin/
2. Click "Login with GitHub"
3. Authorize the app (one-time setup)
4. You're in!

### 3. Create a New Post

1. Click the "New Posts" button
2. Fill in the form:
   - **Title**: Just for the filename (not displayed on site)
   - **Publish Date**: Automatically set to now (can change if needed)
   - **Neocities Era**: Check this box if it's an old post
   - **Body**: Write your post content (supports Markdown)
3. Click "Publish"
4. Done! Your post is live in ~1 minute

### 4. What Happens Behind the Scenes

When you hit "Publish":
1. Sveltia CMS commits your post to GitHub
2. GitHub triggers Cloudflare Pages to rebuild
3. Your site updates automatically (usually 30-60 seconds)
4. Post appears on enrichmetwo.win

### 5. Mobile Tips

**iOS (iPhone/iPad):**
1. Open Safari and go to https://enrichmetwo.win/admin/
2. Tap the Share button
3. Tap "Add to Home Screen"
4. Name it "Post to Site" or similar
5. Now you have an app icon!

**Android:**
1. Open Chrome and go to https://enrichmetwo.win/admin/
2. Tap the menu (three dots)
3. Tap "Add to Home Screen"
4. Name it "Post to Site" or similar
5. Done!

### 6. Writing Your Post

The Body field supports **Markdown**:

```markdown
# Heading
## Subheading

**bold text**
*italic text*

- Bullet point
- Another point

[Link text](https://example.com)

![Image](https://url-to-image.com/image.jpg)
```

### 7. Troubleshooting

**Login not working?**
- Make sure you're logged into GitHub in your browser
- Check that popups are allowed

**Post not appearing?**
- Wait 1-2 minutes for the build to complete
- Check GitHub repo to confirm post was committed

**Can't access admin panel?**
- Verify you're going to https://enrichmetwo.win/admin/ (with the /admin/)
- Clear browser cache and try again

## That's It!

You're ready to post from anywhere. Just bookmark https://enrichmetwo.win/admin/ and you're good to go.

**100% Free. No App. No Clunky Workflow. Just Post.**
