# The Enrichment Hub

Welcome to my personal website repository! This is the home of my online presence where I share projects, thoughts, and whatever else I'm working on.

## About

This is a custom-built personal website featuring a clean, modern design with full dark/light theme support. Built with vanilla HTML, CSS, and JavaScript for maximum performance and simplicity.

## Features

- **Dark/Light Mode Toggle**: Seamless theme switching with dark mode as the default
- **Responsive Design**: Works great on desktop and mobile devices
- **Clean Navigation**: Easy-to-use navigation system across all pages
- **Unified Styling**: Consistent CSS design language throughout the site

## Project Structure

The site is built using static HTML pages with shared CSS styling for a cohesive look and feel. Currently includes pages for projects, information, and other content.

## Development

This is a static website hosted on Cloudflare Pages. Feel free to explore the code and see how it's built!

## Status

Currently under active development with new features and content being added regularly.
